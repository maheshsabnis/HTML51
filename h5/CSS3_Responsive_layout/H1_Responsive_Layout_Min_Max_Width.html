Responsive layout with min and max width

The HTML 5 and CSS 3 have provided various tags and styles respectively for RICH UX over Web. As consumers are in need if RICH UI apps for devices (Mobile,Tablets, etc), it is very important for the developer community to provide such features which should support all browsers (most of them) and easily should be available on the with devices of all screen dimensions.
Since HTML 5 and CSS 3 are technology independent, it can be used along with various technologies like, PHP, JSP(JAVA), ASP.NET, ASP.NET MVC. As of today there are various frameworks are available for RICH Client Side development like jQuery, Knockout.js, AngularJS, Backbone.js etc.
