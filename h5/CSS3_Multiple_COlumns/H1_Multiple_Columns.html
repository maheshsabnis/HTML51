The HTML 5 and CSS 3 is the new web development standard. It is a fast mechanism for developing RIC UX over web. The feature provided in HTML 5 allows developer to develop application compatible to all browsers and also on mobiles. There are several new facilities offered like WebSocket, WebWorker, Using Drad-Drop, Animations, Form Elements, and many more. Offcourse for different browsers some browser specific styles must be used. The demonstration currently uses multi-column feature of the CSS3.
